
# 🛠️ 자격증 기출문제 스크래핑 및 앱 빌드 도구 (Kinz 버전)

이 폴더에는 **Kinz(comcbt.com)** 사이트의 자격증 기출문제를 자동으로 수집하고, 이를 퀴즈 앱에 통합할 수 있는 자동화 도구 세트가 들어있습니다.

## 📂 파일 구성 및 사용 순서

### 1단계: `1_Scraper_Full.py` (데이터 수집기)
- **용도**: 지정된 자격증의 모든 회차 기출문제를 스크래핑합니다.
- **핵심 기능**:
    - **이미지 자동 다운로드**: 문제에 포함된 그림(`.gif`, `.jpg` 등)을 자동으로 감지하여 `images/` 폴더에 저장하고, JSON 파일에는 로컬 경로를 기록합니다.
    - **중복 방지 & 이어받기**: 스크래핑 도중 중단되어도 이미 받은 문제는 건너뛰고 이어서 작업합니다.
    - **지연 시간 설정**: 사이트 차단을 방지하기 위해 각 회차별로 적절한 대기 시간을 가집니다.
- **다른 프로젝트 적용 방법**:
    - 스크립트 상단의 `SUBJECT_URL` 변수를 원하는 자격증 목록 페이지(예: 지적산업기사, 정보처리기사 등)의 URL로 변경하면 됩니다.

### 2단계: `2_Split_Subjects.py` (과목 분류기)
- **용도**: 수집된 전체 문제 파일(`기타.json`)을 문제 번호(1~20, 21~40 등) 기준으로 과목별로 분리합니다.
- **핵심 기능**:
    - **자동 과목 할당**: 보통 20문제 단위로 과목이 바뀌는 기출문제의 특성을 이용하여 자동으로 과목명을 부여합니다.
    - **이미지 경로 유지**: 분류 과정에서도 원본 이미지 링크가 깨지지 않도록 유지합니다.
- **다른 프로젝트 적용 방법**:
    - `SUBJECT_MAP` 딕셔너리의 인덱스(0, 1, 2, 3...)에 해당하는 과목명을 해당 자격증에 맞게 수정하면 됩니다.

### 3단계: `3_App_Integrator.py` (앱 빌더)
- **용도**: 수집/분류된 데이터를 실제 웹 퀴즈 앱(`SurveyApp`) 형식에 맞게 변환하고 통합합니다.
- **핵심 기능**:
    - **데이터 최적화**: 0부터 시작하는 정답 인덱스를 앱용(1부터 시작)으로 변환하고, 앱의 `images` 폴더로 리소스를 복사합니다.
    - **동적 앱 생성**: 앱 실행에 필요한 핵심 파일(`index.html`, `script.js` 등)을 템플릿에서 복사해와서 즉시 실행 가능한 결과물을 만듭니다.

## 💡 특히 유용한 기능: 이미지 처리 (Image Handling)

사용자께서 좋게 보신 **그림 포함 문제 처리**는 다음과 같은 로직으로 작동합니다:

1. **감지**: Selenium이 문제 텍스트 영역 내의 `<img>` 태그를 찾습니다.
2. **저장**: 이미지의 실제 URL을 확인한 후, `requests` 라이브러리를 통해 로컬 `images/` 폴더에 `문제ID_img_0.gif` 와 같은 고유 이름으로 저장합니다.
3. **참조**: JSON 데이터 내의 `local_images` 필드에 해당 로컬 경로를 저장합니다.
4. **표시**: 웹 앱(`script.js`)에서 문제 표시 시 `image` 필드에 값이 있으면 `<img>` 태그를 동적으로 생성하여 화면에 띄웁니다.

## 🛠️ 환경 설정 (Prerequisites)

이 스크립트들을 실행하려면 다음이 설치되어 있어야 합니다:

- **Python 3.x**
- **필수 라이브러리**:
  ```bash
  pip install selenium webdriver-manager requests
  ```
- **Chrome 브라우저**: 최신 필드 크롬 브라우저가 설치되어 있어야 합니다.
