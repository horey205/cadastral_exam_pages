# 📂 프로젝트 분석 보고서: 지적공무원 기출정복 시스템
- **Repository**: https://github.com/horey205/cada_offcie_exam_testing.git

본 프로젝트는 PDF 형식의 기출문제를 디지털 데이터화하여, 모바일 환경에서 학습자가 효율적으로 공부할 수 있도록 구축된 전 과정(추출-정리-학습-관리)을 포함하는 지능형 퀴즈 애플리케이션입니다.

---

## 1. 프로젝트 구조 및 주요 파일

### 🛠️ 데이터 처리 및 유틸리티
*   **데이터베이스 4분할 구조**: 
    *   `official_survey.json` / `official_cs.json`: 과목별 기출문제 원본 데이터.
    *   `custom_survey.json` / `custom_cs.json`: 과목별 AI 생성 및 예상문제 데이터.
*   **`convert_to_js.py`**: 위 4개의 JSON 파일을 병합하여 `questions.js`로 변환하여 브라우저 로딩용 통합 데이터를 생성합니다.
*   **`update_all_answers.py`**: AI와 전문 지식을 결합하여 모든 문항의 정답과 해설을 완벽하게 업데이트하는 스크립트입니다.
*   **`extract_questions.py`**: PDF의 2단 레이아웃을 분석하여 텍스트와 이미지를 자동 추출하는 핵심 엔진입니다.

### 🌐 웹 애플리케이션 (`quiz_app/`)
*   **`index.html`**: 사용자 메인 화면. 학번/이름 입력 절차를 생략하고 즉시 학습 가능한 '공부 중심' UI를 제공합니다.
*   **`script.js`**: '한 문제씩 학습하기(스터디 모드)'와 '20문항 무작위 시험(모의고사 모드)'을 지원하는 핵심 로직입니다.
*   **`admin.html`**: 관리자 전용 대시보드. 문제 수정, 이미지 관리, AI 연동 대량 가져오기 기능을 포함합니다.
*   **`style.css`**: 다크 모드 기반의 글래스모피즘(Glassmorphism) 디자인으로 모바일 가독성에 최적화되어 있습니다.

---

## 2. 주요 기능 및 특징

### 🌟 사용자 학습 모드 (User Side)
1.  **듀얼 학습 모드**:
    *   **학습하기**: 문제를 풀자마자 정답 여부와 상세 해설을 확인하며 빠르게 이론을 정립합니다.
    *   **모의고사**: 20문항을 시간 제한(1분/문항) 내에 풀며 실전 적응력을 높입니다. (난이도 자동 배분: 상4, 중12, 하4)
2.  **모바일 최적화 UX**: 버튼 크기, 이미지 확대, 부드러운 애니메이션 등을 통해 스마트폰 한 손 조작에 최적화된 경험을 제공합니다.
3.  **전문 해설 포함**: 모든 문제에 법령 및 이론 근거를 담은 상세 해설이 포함되어 있어 별도의 해설집 없이 공부가 가능합니다.

### ⚙️ 관리자 시스템 (Admin Side)
1.  **보안 로그인**: 전용 계정(`admin` / `admin205`)을 통한 안전한 데이터 접근.
2.  **지능형 편집기**: 
    *   실시간 이미지 프리뷰 및 로컬 파일 경로 입력 도우미.
    *   문항 내용, 과목, 난이도, 정답의 자유로운 수정.
3.  **NotebookLM 연동 (AI 가져오기)**: 
    *   AI가 생성한 JSON 데이터를 붙여넣기만 하면 수십 개의 문제를 한 번에 추가할 수 있는 **Bulk Import** 기능을 지원합니다.
4.  **확장 가능한 문제은행**: 새로운 문제를 계속 추가하면 모의고사 풀(Pool)이 자동으로 확장되어 중복 없는 문제 출제가 가능합니다.

## 3. 기술적 강점

*   **Offline First**: 서버가 없어도 브라우저만 있으면 모든 기능이 작동하는 분산형 구조입니다.
*   **AI Friendly**: NotebookLM 등 최신 AI 도구가 생성한 데이터를 즉시 수용할 수 있는 개방형 데이터 구조를 채택했습니다.
*   **Image Precision**: 도면 문제가 많은 지적 시험의 특성을 고려하여, 이미지 규격 최적화 및 흰색 배경 가독성 처리가 완벽하게 구현되었습니다.
