# 지적산업기사 기출정복

2017-2020년 지적산업기사 기출문제 퀴즈 앱

## 📚 포함 내용

- 총 1,040개 문제
- 5개 과목별 분류
  - 지적측량 (220문제)
  - 응용측량 (207문제)
  - 토지정보체계론 (199문제)
  - 지적학 (200문제)
  - 지적관계법규 (214문제)

## 📦 데이터 파일

- `questions.js` - 전체 문제 통합 파일 (웹앱용)
- `지적측량.json` - 지적측량 과목 문제
- `응용측량.json` - 응용측량 과목 문제
- `토지정보체계론.json` - 토지정보체계론 과목 문제
- `지적학.json` - 지적학 과목 문제
- `지적관계법규.json` - 지적관계법규 과목 문제

각 JSON 파일은 독립적으로 사용 가능하며, 다른 프로젝트에서도 활용할 수 있습니다.

## 🚀 사용 방법

1. `index.html` 파일을 브라우저로 열기
2. 로그인 (ID: user, PW: user2685)
3. 원하는 과목 선택
4. 학습하기 또는 모의고사 모드 선택
